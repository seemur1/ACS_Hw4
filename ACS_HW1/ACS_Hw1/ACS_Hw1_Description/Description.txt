Полученное Задание:
Реализовать обобщённый массив Различных чисел ( 
Комплексных (действительная и мнимая части – пара действительных чисел)
Простых дробей (числитель, знаменатель – пара целых чисел)
Полярных координат (угол [радиан] – действительное; координаты конечной точки на плоскости))
Реализовать для них функцию приведения каждого значения к действительному числу, эквивалентному записанному. 
Например, для комплексного числа функция реализуется по формуле: sqrt(d^2+i^2)), а для полярных координат - расстояние.
Упорядочить элементы массива по возрастанию используя сортировку методом деления пополам (Binary Insertion). 
В качестве ключей для сортировки и других действий используются результаты функции, общей для всех альтернатив.
